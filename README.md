## 기술 스택

### Frontend
- **React 19.1.1**: 사용자 인터페이스 구축
- **TypeScript**: 타입 안정성
- **Vite**: 빠른 번들링 및 개발 서버
- **Zustand**: 상태 관리

### Backend & Database
- **Java**: 백엔드 애플리케이션 개발 언어
- **Spring Boot**: 백엔드 프레임워크
- **Spring Data JPA**: ORM 기반 데이터베이스 접근 및 관리
- **MySQL**: 관계형 데이터베이스 (데이터 영구 저장)

### 배포 및 호스팅
- **Cloudflare**: 프론트엔드 배포
- **Vultr**: 백엔드 배포
- **GitHub**: 소스 코드 관리
